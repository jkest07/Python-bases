Sistema de Inventario y Facturación en Consola (Python)

1. Requisitos
   - Python 3.8 o superior

2. Instalación
   - Colocar factura.py, inventario.txt y facturas.txt en la misma carpeta.
   - Abrir una terminal en dicha carpeta.

3. Ejecución
   python factura.py

4. Funcionalidades
   INVENTARIO:
     - Añadir producto
     - Listar productos
     - Buscar producto por ID o nombre
     - Actualizar precio o stock
     - Eliminar producto
   FACTURAS:
     - Crear nueva factura (subtotal, IVA 19%, total automáticos)
     - Listar facturas
     - Ver detalle de factura
     - Editar factura
     - Eliminar factura
